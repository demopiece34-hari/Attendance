# Student & Staff Attendance Management System

A Streamlit-based attendance portal with:
- Student login
- Staff login
- Live webcam face verification
- Browser GPS verification
- Google Sheets data storage
- Present / Pending / Absent tracking
- Report generation
- Student enrollment
- Staff-controlled absence marking
